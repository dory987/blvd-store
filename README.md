# BLVD - Moletons Americanos Exclusivos

## Project Description
BLVD specializes in oversized hoodies that embrace the Twitter model, crafted from 100% cotton (fio 30.1). We operate on an encomenda system, ensuring personalized orders tailored to our customers. Our partnership with Aliexpress Boy allows for efficient logistics, with an estimated delivery time of 15-20 business days.

## Features
- Product catalog
- Encomenda system
- Logistics tracking
- Instagram integration
- Newsletter
- Responsive design

## Installation Instructions
To clone and set up the project locally:
1. Clone the repository:
   ```
   git clone https://github.com/dory987/blvd-store.git
   cd blvd-store
   ```
2. Install the necessary dependencies:
   ```
   npm install
   ```

## Usage Guide
- **Viewing Products**: Navigate to the homepage to browse our catalog.
- **Making Orders**: Select the desired product and follow the checkout process.
- **Tracking Deliveries**: Use the logistics tracking feature in your account dashboard.

## Deployment Instructions
You can deploy the project on GitHub Pages by following these steps:
1. Push your code to the `main` branch.
2. Go to the repository settings.
3. Enable GitHub Pages under the "Pages" section. Choose the `main` branch as the source.
